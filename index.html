Projects

PatreonMarker
PatreonMarker is an anti-piracy file hosting service for creators at Patreon. It sends each downloader a unique watermarked content so that when a downloader redistribute his content, the creator can identify him with the content.
dprl: deep reinforcement learning in torch7
dprl is a user-friendly deep reinforcement learning package for torch7. It contains implementation of Google Deepmind's state-of-the-art algorithms: deep Q-learning, double deep Q-learning and asynchronous advanatage actor critic.
Pytorch SSIM
pytorch-ssim is a pytorch implementation of the well-known Structural similarity (SSIM) index. With pytorch's powerful automatic differentiation, we can now develop algorithm that optimize SSIM instead of mean suqare error. The gif image shows the process of optimizing a noise image to the original image.
